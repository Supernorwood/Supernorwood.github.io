= The basic assignment operator is equal ( = ), which assigns the value of its right operand to its left operand. That is, x = y assigns the value of y to x .

== Is a comparison operator in Java checks to see if the two objects being compared take up the same place in memory.

equals() a method available in the String class that is used to compare two strings and determine whether they are equal.

Switch and If/Else Difference

A Switch statementallows a variable to be tested for equality against a list of values. Each value is called a case, and the variable being switched on is checked for each case.

An If statement can be followed by an Else which executes when an expression is false.

It is better to use switch-case instead of if-else when the numbers to of cases to consider are many.If you want more complex conditions to be checked, you want to use if-else.
